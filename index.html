Frontend Mentor Challenges

Frontend Mentor is a website where you can practice and improve your web development skills by working on real projects. They offer a variety of challenges that simulate actual scenarios, making it a great way to learn and grow as a frontend developer.

QR Code Component

A perfect first challenge if you're new to HTML and CSS. The card layout doesn't shift, so it's ideal if you haven't learned about building responsive layouts yet.

3 Column Preview Card

This challenge is perfect if you're just getting started. The shift between the layouts will be a nice test if you're new to building responsive projects.
